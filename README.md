# Numeros_Complejos 🌀

Este repositorio contiene el motor de cálculo para aritmética de variable compleja. Es un módulo de soporte crítico para la investigación de funciones en el plano complejo, transformaciones geométricas y la validación de la **Hipótesis de Riemann**.

## 🚀 Propósito del Proyecto

Proveer una librería robusta de funciones para operar con números complejos ($a + bi$). Este sistema permite realizar las operaciones fundamentales y avanzadas necesarias para la visualización de datos en espacios multidimensionales y el estudio de la distribución de los ceros de la función Zeta.

## 📁 Estructura del Repositorio

Organizado bajo estándares de computación científica:

* **src/**: 
  * Módulos en Visual Basic 6 para operaciones básicas (suma, multiplicación, conjugado).
  * Funciones de conversión entre forma binómica y polar (módulo y argumento).
  * Implementación de la fórmula de Euler y potencias complejas.
* **docs/**: Documentación sobre la aplicación de números complejos en proyecciones estereográficas y mapeos fractales.
* **results/**: Pruebas de cálculo de raíces de la unidad y transformaciones en el plano.

## 🛠️ Aplicaciones Técnicas

* **Visualización de Primos**: Soporte matemático para `Grafica_Primos`, permitiendo proyectar valores sobre la Esfera de Riemann.
* **Fractales y Caos**: Base para el cálculo de conjuntos de Julia y Mandelbrot en el entorno de desarrollo.
* **Geometría Avanzada**: Precursor para el manejo de **Cuaterniones** en rotaciones 3D/4D dentro del proyecto `Figuras_Primos`.
